- La confiance se construit par la **preuve sociale locale** : témoignages, résultats concrets, visages familiers.
- Le prix perçu comme "accessible" doit être ancré sur le **ROI en ventes supplémentaires**, pas sur le coût mensuel.

### 5.3 — Facebook Ads (Budget minimal, impact maximal)

> [!TIP]
> Commencer avec **50 000–100 000 Ar/semaine** en ciblage très précis. Ne pas brûler le budget en large.

**Campagne 1 — Notoriété locale (CPM)**
- Audience : Antananarivo, 25–40 ans, intérêts business/entrepreneur/Facebook pages
- Format : vidéo 30s montrant le bot en action (conversation réelle Messenger)
- Objectif : vues + notoriété marque

**Campagne 2 — Génération de leads (CPL)**
- Format : Lead Ads Facebook (formulaire dans l'app)
- Champs : Nom / Numéro téléphone / Nom de la page Facebook
- CTA : *"Je veux mon chatbot FLARE gratuite 7 jours"*
- Objectif : collecter les contacts qualifiés directement dans Facebook

**Campagne 3 — Retargeting (Conversion)**
- Audience : visiteurs `flareai.ramsflare.com` + interactions page Facebook
- Format : carrousel plans Starter/Pro/Business avec prix MGA
- CTA : *"Activer mon chatbot maintenant"*

### 5.4 — Messenger FLARE comme canal de vente directe

> [!IMPORTANT]
> FLARE AI est un chatbot — **il doit vendre FLARE AI lui-même.**

Le bot Messenger de la page FLARE AI doit être configuré pour :
1. Accueillir les prospects entrants
2. Qualifier (type de business, taille, besoin)
3. Présenter les plans avec prix en MGA
4. Capturer le contact et déclencher le tunnel d'activation
5. Envoyer le lien MVola + instructions paiement

→ **C'est la démo produit la plus puissante possible** : le prospect vit l'expérience en temps réel.

### 5.5 — Groupes Facebook (distribution organique gratuite)

**Groupes prioritaires à cibler :**
- Groupes entrepreneurs malgaches (FIVMPAMA, entrepreneuriat MG)
- Groupes e-commerce / vente en ligne Madagascar
- Groupes commerce / boutiques Antananarivo
- Groupes marketing digital Madagascar

**Règle de posting dans les groupes :**
- Ne jamais poster un pub direct → bannissement
- Poster une valeur (conseil, insight) → mentionner FLARE naturellement
- Répondre à des questions sur la gestion client / Facebook / chatbot = opportunité

## 6. Canal 2 — TikTok (Viralité organique)

### Rôle stratégique

TikTok = **canal de notoriété et viralité gratuite** sur les jeunes entrepreneurs (18–30 ans).  
Pas de conversion directe attendue. Objectif : **faire connaître FLARE à coût zéro**.

> [!NOTE]
> TikTok est en forte croissance à Madagascar. L'algorithme favorise les comptes locaux avec du contenu authentique. Zéro abonné au départ n'est pas un obstacle si le contenu est bon.

### 6.1 — Type de contenu TikTok (3–4 vidéos/semaine)

#### Pilier 1 — "Avant/Après" entrepreneur
```
POV : Tu lances FLARE AI sur ta boutique Facebook
[Avant] → messages non lus, clients qui attendent
[Après] → bot répond à 2h du mat, toi tu vois les leads le matin
```
Format : voiceover + screen recording Messenger + sous-titres

#### Pilier 2 — Démonstration en direct
```
[Screen recording] Je te montre comment FLARE répond à la place d'une boutique malgache
Client : "Bonjour, vous avez le prix du sac modèle X ?"
Bot : [réponse immédiate, qualifiée, avec catalogue]
```
Format : écran + texte rapide + musique tendance

#### Pilier 3 — Éducation business (hook fort)
```
❌ Stop de répondre manuellement à tes DM Facebook
Voici pourquoi tu rates des ventes sans le savoir → [explication courte]
La solution : ton chatbot FLARE AI
```
Format : parler face caméra ou texte animé

#### Pilier 4 — Behind the scenes FLARE
```
Comment on active un chatbot en 15 minutes pour un entrepreneur malgache
[Timelapse cockpit FLARE → configuration → test Messenger → bot actif]
```
Format : POV dev/opérateur + musique

#### Pilier 5 — Social proof malgache
```
[Témoignage client] : "Avant je ratais des messages la nuit. Maintenant mon bot répond."
— [Prénom, type de boutique, ville]
```
Format : interview courte 30s ou texte + photo

### 6.2 — Règles TikTok pour la viralité

- **Hooks** : les 2 premières secondes décident tout → commencer par une question, un chiffre, ou un choc
- **Durée** : 15–45s pour la portée max, 60s max pour l'éducation
- **Sous-titres** : obligatoires (beaucoup regardent sans son)
- **Hashtags** : `#entrepreneurmadagascar #chatbotfacebook #flareai #mvendre #antananarivo #business`
- **Fréquence** : régularité > qualité de production au départ
- **Réponses vidéo** : répondre aux commentaires par vidéo = boost algorithmique

### 6.3 — TikTok → Facebook (le funnel)

```
TikTok (découverte) → Profil TikTok → Lien Bio → Page Facebook FLARE → Messenger → Activation
```

Le bio TikTok doit pointer vers la page Facebook FLARE AI ou directement vers `flareai.ramsflare.com`.

## 11. Risques à surveiller

> [!WARNING]
> **Risque #1 — Surcharge opérateur** : Si les leads arrivent vite et que l'activation reste 100% manuelle, le SLA de 15 min ne tiendra pas. Prévoir dès le mois 1 un protocole de priorisation.

> [!WARNING]
> **Risque #2 — Mauvaise attente client** : Si le message marketing dit "activé instantanément" et qu'il faut attendre 24h, le NPS s'effondre. Être honnête sur le délai assisté.

> [!CAUTION]
> **Risque #3 — Faux départ sans preuve** : Lancer des Ads avant d'avoir 1 client satisfait = argent gaspillé. Faire les 5 premiers clients en direct avant de scaler l'acquisition payante.
