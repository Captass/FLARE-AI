# FLARE AI Frontend Rules

- Default squad here: `creative_director`, `frontend_principal`, `qa_breaker`.
- Add `performance_release_engineer` for animation, bundle size, sluggish rendering, or complex responsiveness.
- Preserve the established product language unless the user explicitly asks for a redesign.
- Keep desktop and mobile both strong.
- Prefer system-level fixes over local UI hacks.
- Verify loading, empty, error, locked, and permission states for user-facing flows.
- Run `npm run build` in `frontend/` after meaningful frontend changes.
- Update docs if a visible flow or setup changes.
